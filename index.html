Word Guess Game

This game is very similar to the traditional Hang Man game. Select any letter using your keyboard to guess a letter. If the letter is correct, it will fill in the appropriate "_" - If you give an incorrect answer though, you will lose a life! You are only given 5 lives at the start of each word - so choose wisely!

To begin the game simply select the first letter you would like to play. Good Luck!
